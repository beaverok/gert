Компания WSS-Consulting специализируется на внедрении продуктов Microsoft Office Servers: SharePoint Server 2007 / 2010, SharePoint Services 3.0, SharePoint Foundation 2010, Search Server 2007 / 2010, FAST Search Server 2010.

Преимущества работы с нашей компанией:

  • Сильная эспертиза по платформе SharePoint
  • Большая команда программистов, аналитиков и инженеров, специализирующихся на данной платформе
  • Наличие готовых решений и продуктов
  • Конкурентоспособная стоимость продуктов и услуг
  • Экспертиза в предметных областях: документооборот, корпоративные порталы
  • Отлаженная методология, гарантирующая высокое качество продуктов и услуг

история

Компания WSS-Consulting создана в середине 2006 года двумя основателями – экспертами в области разработки корпоративных информационных систем. Основой для компании стала веб-студия, создававшая интернет-сайты с 2003 года.

Ключевыми идеями при создании компании WSS-Consulting были:

  • Специализация на платформе Microsoft SharePoint
  • Ориентация на создании качественных промышленных решений в области корпоративных порталов и автоматизации документооборота

С первых месяцев работы компания заслужила репутацию наиболее технологически продвинутого игрока рынка внедрения SharePoint. Рекомендации клиентов послужили катализатором, позволившим компании динамично развиваться, увеличивая штат компании и финансовые показатели не менее чем на 100% каждый год.

К середине 2013 года в компании WSS-Consulting работает более 50 специалистов по SharePoint, SQL Server и Microsoft.Net. Мы успешно выполнили более 100 проектов внедрения SharePoint для средних и крупных заказчиков.

Компания продолжает развиваться, и мы ставим себе цель: до 2014 года стать игроком №1 российского рынка продуктов и услуг на платформе Microsoft SharePoint.
